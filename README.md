# Aquarium

Браузерная игра-аквариум на [PixiJS](https://pixijs.com/): живая обложка с рыбками и match-3 внутри.

**Играть:** https://shurygina-jul.github.io/aquarium/

## Что внутри

**Главный экран** — фон, каустика, рыбы и пузырьки. Кнопка «Играть» открывает match-3.

**Match-3** — поле 8×8, счёт, кнопка «Назад». Пока идёт анимация, поле не реагирует на ходы.

## Как играть

### Обычный ход

Можно **кликнуть** две соседние плитки по очереди или **свайпнуть** одну в сторону соседа. Диагонали не работают.

Если после обмена нет совпадения — плитки возвращаются. Если есть — они исчезают, остальные падают вниз, сверху появляются новые. Если снова складывается ряд — каскад продолжается.

Считаются только горизонтальные и вертикальные линии из трёх и более одинаковых плиток. Стартовое поле собирается без готовых троек.

### Бонусы

| Что собрать | Что появляется | Как работает |
| ----------- | -------------- | ------------ |
| Квадрат 2×2 | Серебряный алмаз | Срабатывает сам: летит к случайной плитке, обе исчезают, дальше падение и каскад |
| 5+ связных плиток одного типа | Золотой алмаз | Свайп к соседней обычной плитке — или клик по алмазу, затем клик по соседу. Убирает все плитки этого типа на поле |

Если на поле одновременно подходят несколько паттернов, золотой бонус важнее серебряного.

Повторный клик по золотому алмазу в режиме выбора отменяет его.

### Очки

+1 за каждую убранную плитку — в том числе в каскадах и при срабатывании бонусов.

## Разработка

```bash
yarn install
yarn dev      # локально
yarn build    # сборка
yarn preview  # просмотр dist/
yarn lint     # eslint
```

Стек: TypeScript, Vite, PixiJS 8.

Код экранов лежит в `src/screens/` (`cover/` — обложка, `matchThree/` — игра). Переключение сцен — `sceneManager.ts`. Общий фон — `src/shared/`, кнопки поверх canvas — `src/ui/htmlButtons.ts`.

Пуш в `main` публикует демо через [GitHub Actions](.github/workflows/deploy.yml).
